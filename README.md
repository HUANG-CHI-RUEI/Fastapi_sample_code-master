## Quick Start
- Step1 Create python virtual environment
 ```bash
 python -m venv .venv
 .venv/Scripts/Activate.ps1
 ```

- Step2 install package
 ```bash
 pip install -r requirements.txt
 ```

- Step3 Run FastAPI
 ```python
 uvicorn main:app --reload
 ```
